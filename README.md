# 👟 Nike Sales Analysis – Power BI Project

## 📖 Project Description
The Nike Sales Analysis project focuses on analyzing and visualizing sales data to gain insights into business performance. Using Power BI, this project transforms raw sales data into interactive dashboards that help track revenue, profit, and customer trends.

## 🎯 Objectives
- Analyze sales performance across regions and categories
- Identify top-selling products
- Monitor revenue and profit trends
- Understand customer purchasing behavior

## 🛠️ Tools & Technologies
- Power BI
- Power Query (Data Cleaning)
- DAX (Data Analysis Expressions)
- Excel / CSV Dataset

## 📊 Key KPIs
- Total Sales Revenue
- Total Profit
- Total Orders
- Profit Margin (%)
- Quantity Sold

## 📈 Dashboard Features
- Sales by Region (North, South, East, West)
- Sales by Product Category (Shoes, Apparel, Accessories)
- Monthly and Yearly Sales Trends
- Top 10 Best-Selling Products
- Customer Segmentation Analysis

## 📉 Visualizations Used
- Bar Chart (Sales by Category)
- Line Chart (Revenue Trend)
- Pie Chart (Category Distribution)
- KPI Cards (Key Metrics)
- Map Visualization (Regional Sales)

## 🔍 Insights & Analysis
- Identified highest revenue-generating regions
- Found top-performing products
- Analyzed seasonal sales trends
- Detected low-performing categories

## 🚀 How to Use
1. Load dataset into Power BI
2. Clean data using Power Query
3. Create relationships between tables
4. Build KPIs using DAX
5. Design interactive dashboard
